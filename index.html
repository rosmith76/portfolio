Bike Maintenance Tracker

Single page app created with Ember.js. The backend API was built with Ruby on Rails. I created this app as a passion project. I am an avid cyclists. While I was at GA I commuted by bike, and I ride my mountain bikes on the weekends. With 4 bikes I needed a way to keep track of simple maintenance issues for each bike. Bikes are similar to code in that a small part failure can put the bike out of commission. In the future I would like to allow user image uploads, and I would like to add background images that change with the view state.
